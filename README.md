# Personal Web Page

Tugas Praktikum Rekayasa Perangkat Lunak  
Topik: Dasar Pengembangan Web (HTML)

## Deskripsi
Project ini merupakan halaman web sederhana berupa Personal Web Page (Halaman Profil Pribadi) yang dibuat menggunakan HTML.

## Fitur Halaman
- Biodata lengkap
- Riwayat pendidikan (tabel)
- Daftar keahlian (technical & non-technical skills)
- Gambar profil
- Form kontak sederhana
